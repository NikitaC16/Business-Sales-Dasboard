# Business Sales Performance Dashboard

## Project Overview
This project analyzes business sales performance data from 2014–2017 using Microsoft Excel.

The dashboard helps identify:
- Revenue trends over time
- Most profitable regions
- Top-performing products
- Customer segment contributions

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Analysis

---

## Key KPIs
- Total Revenue: R2 295 273,92
- Total Profit: R285 988,28
- Profit Margin: 12,46%

---

## Key Insights
- Revenue increased steadily from 2014 to 2017.
- West region generated the highest profit.
- Phones and Chairs were the top-performing products.
- Consumer customers contributed the highest share of revenue.

---

## Business Recommendations
- Focus marketing efforts on high-performing regions.
- Increase promotion of top-selling products.
- Improve profitability in lower-performing regions.
- Develop strategies targeting Consumer customers.
